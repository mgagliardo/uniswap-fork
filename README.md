# Homework 14

## How to run this.

1. Clone the repo
2. Run the hardhat node:
```
$ npx hardhat node
```
3. Run scripts (deploy contracts).
```
npx hardhat run scripts/script.js 
```
4. Run tests
```
npx hardhat test --network hardhat
```
